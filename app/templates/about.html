Mission & Engineering Goals

The Customer Segmentation Platform is an enterprise-grade AI analytics system built to analyze credit card customer behaviors, construct quantitative segment taxonomies, and formulate McKinsey-style business strategies. Designed to exceed standard benchmarks, it combines scalable unsupervised machine learning with a full-stack REST API and interactive dashboard interface.

Technology Stack Matrix

Backend Core
  • FastAPI — High-performance REST web framework
  • Python 3.11/3.12 — Core programming language
  • SQLAlchemy & SQLite — Execution logging & database
  • Pydantic v2 — Data validation & settings
Machine Learning
  • Scikit-Learn — K-Means, Agglomerative, PCA, Scaler
  • Pandas & NumPy — Data manipulation & log1p transform
  • SciPy — Ward linkage hierarchical clustering
  • Plotly & Chart.js — Interactive 2D/3D visualizations
Frontend UI/UX
  • Jinja2 Templates — Dynamic HTML rendering
  • Bootstrap 5.3 — Responsive CSS framework
  • Glassmorphism CSS — Custom Dark/Light theme
  • Font Awesome 6 — Scalable vector icon suite
DevOps & Deployment
  • Docker & Compose — Multi-stage containerization
  • Nginx — Production reverse proxy & caching
  • Render / Railway — Cloud app hosting
  • GitHub Actions — Continuous integration pipeline
REST API Endpoints
